# Flutter Assets

This folder contains Flutter assets.

## Launch Images

To use your own launch image:
1. Add images to `Assets.xcassets/LaunchImage.imageset`
2. Update `LaunchImage.imageset/Contents.json` with the correct filenames
3. Or use the Flutter CLI to generate them:
   ```bash
   flutter pub run flutter_launcher_icons:main
   ```

## App Icons

To use your own app icon:
1. Add images to `Assets.xcassets/AppIcon.appiconset`
2. Update `AppIcon.appiconset/Contents.json` with the correct filenames
3. Or use the Flutter CLI to generate them:
   ```bash
   flutter pub run flutter_launcher_icons:main
   ```
